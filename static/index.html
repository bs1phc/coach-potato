👋 Welcome to Coach Potato! Add your Riot API key and the accounts to track, save, then hit Update data to fetch your match history (a long first crawl — roughly 2 minutes per 100 games).

Riot API & accounts

Get one at developer.riotgames.com. Best option: register a personal product there for a persistent key. The instant development key works too, but expires every 24 h — paste a fresh one here when crawls start failing.

Any number of Riot IDs, e.g. Player#EUW. Stats combine all of them.

While the app is open, new games are fetched on this schedule (0 = off). Data also updates once on startup and whenever you press Update data.

Blocks

Blocks auto-advance after this many games. Changing it applies to the current and future blocks; finished blocks keep their size.

Blocks are meant to be played in succession: if a game you add starts this many hours (game time) away from the block's latest game, the block closes and a new one starts. 0 turns this off.

Untick to auto-close silently when the time gap is exceeded.

Export everything (.zip)

Every coaching session, block learning, matchup guide, champion note, item build, and Research entry you've written — plus every uploaded clip and screenshot — as one .zip file. Doesn't include your API key/accounts or crawled match history (Riot's API can always re-supply that).

Restore an "Export everything" .zip — only meant for a fresh/empty setup (new machine, reinstall). Refuses (nothing is written) if anything in the backup would collide with data already here.
